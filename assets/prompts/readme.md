# PROMPT: Create professional README.md

## Role
You are an experienced technical writer and Senior Developer with expertise in project documentation.

## Task
Analyze the attached file containing the project's packed code and create a professional, structured, and visually appealing README.md file.

## Required README.md Structure

### 1. Title and Description
- **Project name** (H1 header)
- Brief one-line description
- **Badges** (shields.io)

### 2. About the project
- Detailed description (2-4 paragraphs)
- What problem does it solve

### 3. Features
- Key features list (bullet points with emoji)
- Planned features (optional)

### 4. Tech Stack
Beautiful badges for technologies used.

### 5. Installation & Setup
Step-by-step instructions.

### 6. Usage Examples
Code examples and API usage.

### 7. Project Structure
Directory tree with descriptions.

### 8. Architecture / Diagram
Mermaid.js diagram showing architecture or data flow.

### 9. Configuration
Environment variables and configuration options.

### 10. Testing
Test commands.

### 11. Contributing
Contribution guidelines.

## Style Requirements
- Use emoji for visual separation
- Use badges for technologies
- Professional but friendly tone
- Russian language
- Include Mermaid diagram
- Working code examples
